## 笔记格式

每篇笔记包含：
- 论文基本信息（标题、作者、年份、关键词）
- 研究背景与问题
- 核心贡献 / 创新点
- 方法与模型详解
- 实验与结果
- 结论与局限性
- 难懂概念的比喻解释（💡 标注）
